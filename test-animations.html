Infinity Storm - Animation Test

Character Animation Tests

Open the game and use these keyboard shortcuts to test:

W - Test Scarlet Witch idle/attack animations

T - Test Thanos idle/attack animations

Both animations will play idle first, then attack after 3 seconds, then return to idle.


To Test Random Multiplier with Character Animations:

1. Play the game normally until you get a win

2. Random Multiplier has a chance to trigger after wins

3. When it triggers, it will randomly choose between Thanos or Scarlet Witch

4. The chosen character's attack animation will play during the multiplier effect


Note: Make sure to run the game from the server (npm start in infinity-storm-server)
